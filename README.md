# Rime Weasel 小狼毫 配置文件
Rime真是太好用啦 🥰
## 主要功能
+  明月拼音-简体字
+  中英文输入
+  emoji
+  常用语语言模型
+  以词定字
## 使用
1. 安装 Rime Weasel 小狼毫
2. 将本项目文件替换到用户文件夹（可以先备份原用户文件夹）
## 测试
+ 英文输入、表情
+ "，""。"号选择最左最右字符 => 以词定字
+ 充滿希望的跋涉比到達目的地更能給人樂趣 => 常用语语言模型
